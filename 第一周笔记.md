### Python 的数据操作库

* Pandas
* SFrame（课程选用）

### 选用原因

* SFrame 和 GraphLab Create 都是高度可伸缩的（highly scalable）；
* SFrame 包含在 GraphLab Create 中，不必下载多个包；
* SFrame 不局限于适合内存的数据集，因此可以在笔记本电脑上处理大型数据集。
